# SAAO
 Objetivo: Disponibilizar informações de obras públicas para gestores públicos e sociedade civil de forma estruturada e acessível, possibilitando maior efetividade de participação dos diversos agentes da sociedade no acompanhamento e orçamento de obras em suas cidades, utilizando os recursos tecnológicos existentes como ferramenta de inclusão e apoio no orçamento participativo.
